# 📈 Real-Time Financial Market Stock Tracker & Analytics Dashboard

A professional, low-latency financial market tracking dashboard and business intelligence platform built using **Python**, **Yahoo Finance Core API (`yfinance`)**, **Pandas**, and **Plotly**, wrapped in a responsive **Streamlit** user interface. 

This application streams live asset pricing structures directly from international market tickers, executes math pipelines to inject rolling technical moving averages, and renders interactive analytics frames for strategic financial mapping.

## ✨ Core Engineering Features
* **Live Market Ticker Streaming:** Intercepts real-time trading attributes (Open, Close, Session Volatility, Volume) by orchestrating asynchronous calls to global financial server nodes.
* **Technical Moving Average Indicators:** Implements mathematical data window transformations via Pandas rolling window matrices to compile 20-day and 50-day asset averages.
* **Dynamic Hardware-Accelerated Charts:** Renders fluid, dark-mode financial Candlestick and Trend Line charts using Plotly, complete with structural range-slider zoom controls.
* **Transactional Ledger Engine:** Transforms time-series pandas structures into organized consumer data frames for direct download and row auditing.

## 🛠️ Technology Stack
* **Financial Core Integration Layer:** Yahoo Finance yfinance API Node
* **Data Cleansing & Math Pipeline:** Pandas, NumPy
* **Interactive Visualization Engine:** Plotly Graph Objects (Interactive Candlestick Matrices)
* **Microserver Deployment Interface:** Streamlit Interface Engine
